# QUEUEBOOK v1.0

A simple Tkinter application for tracking walk-in customers in a FIFO queue.

## Features
- Add customers to queue
- NEXT customer system (FIFO removal)
- Queue display using Treeview
- Input validation (prevents empty entries)

## Technologies
- Python
- Tkinter
- ttk.Treeview
